使用photoshop24.5.0将照片底色改为纯白色

发表于 2024-10-21 分类于 技术兴趣 ， 图片处理 Valine：
本文字数： 153 阅读时长 ≈ 1 分钟
  1. 打开 Adobe Photoshop 24.5.0 软件；
  2. 文件 => 打开 => 选择要处理的图片；
  3. 点击图片下方的”移除背景”；

    移除背景后的图片如下:
  4. 点击右下角”创建新图层”；
  5. 选择”油漆桶工具”，然后选择底色照片(这里是白色)，然后点击图片，就增加了纯白的图层；

  6. 将新建的图层移到原图层下方，就得到了白底的照片。
