Windows11 激活方法

发表于 2024-02-26 更新于 2025-04-16 分类于 技术兴趣 ， 软件应用 Valine：
本文字数： 402 阅读时长 ≈ 1 分钟

今天将笔记本自带的 win11 家庭版升级到专业版时无意中在知乎中找到了一个方法，一试还真的有效，因为远程运行了一个网站的程序，不知道自己的电脑是不是成为别人的肉鸡了，但是激活方式确实是有效的。风险性后面再研究吧。

两个激活秘钥

我使用的是第一个，第二个是否有效没有验证。

方法

  1. 右键“此电脑”，依次点击“属性” => “产品秘钥和激活” => “更改产品秘钥”，输入上述秘钥；
  2. win + R，输入 powershell，回车。在弹窗中输入命令:irm massgrave.dev/get.ps1 | iex
  3. 会弹出下面的窗口

    简单解释一下: 按 1，Windows 永久激活；按 2，office 永久激活；按 3，Windows 激活到 2038 年；按 4，180 天的 kms 激活工具；……
  4. 然后重复 1 就会看到“已激活”字样。
