使用 curl 命令行工具查询本地公网 IP 地址

发表于 2024-02-04 更新于 2025-04-16 分类于 技术兴趣 ， 网络服务 Valine：
本文字数： 1.8k 阅读时长 ≈ 2 分钟

在 Linux 中 curl 是一个利用 URL 规则在命令行下工作的文件传输工具，同时也可以查询本机公网 IP 地址，具体使用方法如下:

  1. 使用 curl ipinfo.io 查询:
    curl ipinfo.io

    返回值
  2. 使用 curl ifconfig.me 查询 
    curl ifconfig.me

    返回值
  3. 使用 ipinfo.io 加ip地址，可查血这个 ip 的信息 
    curl ipinfo.io/192.168.32.84

    返回值
  4. 此命令在 Windows 的命令提示符中也可以使用
    使用方法如下:
  5. 最简单的是使用 https://ip138.com/ 网址进行本机公网 IP 地址查询
