在 Windows11 中找回照片查看器打开图片

发表于2023-12-12|更新于2025-04-16|技术兴趣软件应用
|浏览量:
  1. 使用快捷键 Win + R 打开运行窗口， 输入 regedit 命令回车打开注册表
  2. 找到HKEY_LOCAL_MACHINE -> SOFTWARE->Microsoft -> Windows Photo Viewer -> Capabilities -> FileAssociations，右键 => 新建 => 字符串值，名称填写.jpg，数值填写PhotoViewer.FileAssoc.Tiff，同理添加 .png .jpeg，数值相同。
  3. 添加之后就可以使用 Windows 照片查看器看图。
